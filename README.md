# Features

## Messaging

- Realtime messaging.

## User Profiles

- Edit profile.
- Follow / Unfollow people.
- add / delete posts.

## Posts

- Like posts.
- Comment on posts.
- Share posts.

## Notifications

- get notifications from other users.
- likes / comments / follows.

## Search

- search for other users.
- updates as you type.

## Themes

- light / dark mode.

## Mobile

- built for mobile as well.
